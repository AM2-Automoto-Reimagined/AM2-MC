  
  

# AM2-MC Server Modpack

  

For game version 1.16.5 and Forge version 36.2.42

  

## Mod installation

  

Copy `<mods>` folder to the `<game root>` directory (exp. `C:\Users\User\AppData\Roaming\.minecraft`)
